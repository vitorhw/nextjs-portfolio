solidarity - allows users to support NPOs and follow their actions through this social media-like platform.

## Features

- **Authentication:** SSO authentication with Google Account
- **Contribute:** Curated list of NPOs in the homepage, users can make a one-time or recurring donation

![user view](https://github.com/vitorhw/nextjs-portfolio/blob/main/public/projects/solidarity/screenshot.png?raw=true)

## Built with

- [React](https://react.dev/)
- [NestJS](https://nestjs.com/)
- [Stripe](https://stripe.com/)
- [Tailwind](https://tailwindcss.com/)
- Deployed on [Vercel](https://vercel.com/)

## Running the project

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `pages/index.js`. The page auto-updates as you edit the file.
